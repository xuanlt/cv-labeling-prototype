dvc add data/raw/ dvc push

1
Copy ảnh thẳng vào thư mục server bằng rsync:
# Đồng bộ toàn bộ thư mục ảnh rsync -avz --progress \ ./images/ \ cvlabel@10.0.1.45:/mnt/cvlabel/projects/lp-detection-v2/raw/
2
Hoặc dùng scp cho file đơn lẻ:
scp batch_night.zip \ cvlabel@10.0.1.45:/mnt/cvlabel/projects/lp-detection-v2/raw/
Upload ảnh qua REST API. Yêu cầu API key từ trang cài đặt tài khoản.
# Upload một file curl -X POST \ https://cvlabel.internal/api/v1/projects/proj_lp_v2/images \ -H "Authorization: Bearer YOUR_API_KEY" \ -F "file=@cam_001.jpg" # Upload batch (zip) curl -X POST \ https://cvlabel.internal/api/v1/projects/proj_lp_v2/images/batch \ -H "Authorization: Bearer YOUR_API_KEY" \ -F "archive=@batch.zip"
